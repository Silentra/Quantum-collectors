## SciCards - Physics & Science Card Collecting Game

**Concept:** Educational card game where students collect scientist and science concept cards through pack opening. Admin-controlled access codes, groups, and live config.

**Core Mechanic:** Access-code-gated registration, pack opening with weighted rarity odds, admin dashboard for full game control.

**Art Style:** Clean dark-mode UI with Tailwind CSS, rarity-colored card borders, responsive student-friendly layout.

**Phase 3:** Extended card schema (imageUrl, keyFact, auraType, enabled) with full admin CRUD editor. Redesigned player-facing card renderer with bordered frames, artwork images, field/keyFact layout, rarity-distinct styling, and an aura tier system derived from duplicate ownership count. Legacy fields preserved for backward compatibility.
